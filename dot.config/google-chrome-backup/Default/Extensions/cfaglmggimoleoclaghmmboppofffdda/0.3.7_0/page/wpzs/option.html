百度网盘
在 网盘主页、 网盘时光轴 显示【生成链接】按钮。
在 多文件分享页、 单文件分享页 显示【生成链接】按钮。
在 带提取码分享页[eqh1] 自动填充提取码。
如果没有查询到提取码，验证成功会自动分享提取码。
分享个人网盘的文件时，可自定义分享的提取码。
在 文件分享页 显示分享的来源列表。
在 生成多文件链接 的弹窗显示批量链接的文本框。
查询并填充提取码后，自动提交提取码并跳转页面。
更多请查看 插件主页。
记得定时 清理临时文件，减少网盘空间占用。
